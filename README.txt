src/ contains source code.
tutorial/ contains simple example files to be used in conjunction to the online tutorial.
bin/ is used to contain compiled executables.

Please refer to the change log at http://alchemy.cs.washington.edu/lite/
for the latest changes to Alchemy Lite.

To make, just type "make" or "make al" at the command line.

To run, type:

> ./bin/al <arguments>

If and when you find bugs, please send them to alchemy-lite@cs.washington.edu.
